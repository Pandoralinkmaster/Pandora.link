# APK automatisch bauen lassen

## So bekommst du die fertige Pandora.apk ohne eigenen PC-Aufwand:

### 1. GitHub-Account anlegen (falls noch nicht vorhanden)
→ https://github.com/signup (kostenlos)

### 2. Neues Repository anlegen
→ https://github.com/new
- Name: `pandora`
- Privat ✓ (wichtig – das ist dein System)
- Repository erstellen

### 3. Dateien hochladen
- Im neuen Repo auf **"uploading an existing file"** klicken
- Den Inhalt des `pandora-android/` Ordners hochladen
- Commit: "Initial commit"

### 4. GitHub baut automatisch die APK
- Tab **"Actions"** öffnen
- Der Build läuft (~5-8 Minuten)
- Grüner Haken = fertig

### 5. APK herunterladen
- Build anklicken → unten **"Artifacts"**
- **Pandora-APK** herunterladen → `Pandora.apk`

### 6. Auf dem Samsung installieren
- APK per USB oder per Handy-Browser herunterladen
- Tippen → **Installieren**
- Fertig ✓
